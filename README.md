# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website developed using HTML5, CSS3, and JavaScript. It showcases my skills, projects, education, internship experience, resume, and contact information in a modern and accessible design.

The portfolio is designed with a clean user interface, smooth animations, responsive layouts, dark/light theme support, and accessibility best practices.

## Features

### Personal Introduction
- Hero section with introduction and career goals
- Typing text animation
- Internship availability badge

### About Section
- Educational background
- Career objectives
- Quick facts and current learning activities

### Skills Section
- Web Development
- Programming Languages
- AI & Machine Learning
- Development Tools

### Projects Section
- Image Gallery Website
- Task Manager Application
- Student Portfolio Website
- AI-Powered Web App (In Progress)

### Resume & Experience
- Frontend Development Internship
- AI & ML Learning Program
- Education Timeline
- Resume Download Option

### Contact Section
- Contact form with validation
- Responsive design
- Accessible navigation

### Additional Features
- Responsive Design
- Dark/Light Theme Toggle

- Scroll Reveal Animations
- Hover Effects
- Mobile Navigation Menu
- Accessibility (WCAG & ARIA)
- SEO Meta Tags
